Una interfaz de usuario (también conocida como UI, por sus siglas en inglés de User Interface) es el medio a través del cual un usuario interactúa con un sistema o una máquina. Puede ser tanto física, como una pantalla táctil, un teclado o un mouse, como virtual, como una interfaz gráfica en una aplicación de software.

El objetivo principal de una interfaz de usuario es permitir que el usuario interactúe de manera efectiva y cómoda con el sistema o la máquina. Esto implica proporcionar elementos visuales y funcionales que permitan la entrada de datos, la navegación, la visualización de información y la ejecución de acciones.

Algunos ejemplos comunes de interfaces de usuario incluyen:

  • Interfaz Gráfica de Usuario (GUI): Es una forma de interfaz que utiliza elementos gráficos como botones, menús, ventanas y otros controles visuales para permitir la interacción del usuario con un programa de software. Los sistemas operativos modernos, como Windows, macOS y muchas aplicaciones de software, utilizan interfaces gráficas de usuario.
  • Interfaz de Línea de Comandos (CLI): En contraste con una GUI, una CLI permite a los usuarios interactuar con un sistema mediante el uso de comandos de texto. Los usuarios escriben comandos específicos para realizar acciones o acceder a funciones. Los sistemas operativos Unix y Linux, así como las bases de datos y algunas aplicaciones, a menudo utilizan interfaces de línea de comandos.
  • Interfaz de Pantalla Táctil: Se utiliza comúnmente en dispositivos móviles, cajeros automáticos, quioscos de información y otros dispositivos interactivos. Permite a los usuarios tocar la pantalla con los dedos o con un lápiz óptico para realizar selecciones o acciones.
  • Interfaz de Voz: Permite a los usuarios interactuar con un sistema mediante comandos de voz. Los asistentes virtuales como Siri, Google Assistant y Amazon Alexa son ejemplos de interfaces de voz.
